# 🍽️ Carta QR – La Trattoria (Demo)

Una carta digital interactiva tipo QR para restaurantes, bares y cafeterías.  
Diseñada para ser rápida, visual y optimizada para móvil.

👉 Demo en vivo: https://marfdezc.github.io/menu-qr/

---

## ✨ Qué incluye

- 📱 Diseño 100% responsive (móvil primero)
- 🍕 Menú por categorías (pizzas, pasta, bebidas, etc.)
- 🖼️ Fotos por plato con visualización en modal
- ⚡ Interacción rápida tipo app
- 👆 Indicador de “tap para ver imagen”
- 🎨 UI moderna, limpia y fácil de usar

---

## 🚀 Tecnologías

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript vanilla
- Font Awesome

---

## 💡 Idea del proyecto

Este proyecto simula una carta QR real que podría usar un restaurante físico.

La idea es mostrar cómo un negocio puede:
- Digitalizar su carta
- Evitar papel
- Mejorar la experiencia del cliente
